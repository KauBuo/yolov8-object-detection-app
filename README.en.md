# yolov8目标检测APP

#### Description
本项目是一个基于Tkinter和OpenCV的目标检测应用程序，实现了摄像头和视频文件的实时目标检测。通过YOLOv8模型进行目标检测，支持定位、分割和姿势三种模型类型，以及不同模型大小。
